## Description
This is a simple library for **TB6612 motor** driver under the **STM32 HAL** framework.
## Features
- **Set Motor Speed** using PWM
- **Control Motor Direction** (Forward & Backward)
- **Brake & Standby Mode**
- **Compatible with STM32 HAL**
